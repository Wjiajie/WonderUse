# 妙物记 (MiaoWuJi) — 产品需求文档 (PRD)

> **版本**: v1.0 MVP
> **创建日期**: 2026-03-28
> **状态**: 已评审，待开发

---

## 1. 产品概述

### 1.1 一句话描述

**妙物记是一个拟物风格的探索型 Web 应用——你的好奇猫咪"喵呜"会陪你一起发现手头产品的隐藏宝藏，让每一件物品都被充分热爱。**

### 1.2 核心理念

每个产品都有"两次消费"：

- **第一次消费**：购买产品
- **第二次消费**：发现并利用产品的全部功能和优势

妙物记通过快乐、有成就感的方式引导用户做更多"二次消费"，从而减少不必要的重复购买和浪费。

### 1.3 产品名称 & 吉祥物

| 项目 | 内容 |
|------|------|
| **中文名** | 妙物记 |
| **英文名** | MiaoWuJi |
| **吉祥物** | 喵呜 (MiaoWu) — 一只银色虎斑美短猫 |
| **Slogan** | "每一件物品，都值得被热爱" |

## 2. 目标用户

### 2.1 用户画像

| 维度 | 描述 |
|------|------|
| **年龄** | 25-40 岁 |
| **特征** | 有消费意识的年轻人、关注可持续消费、喜欢整理和记录 |
| **痛点** | 买了很多东西却只用到 20% 的功能，总想买新的来解决问题 |
| **需求** | 一个有趣的方式来重新发现手头产品的价值 |
| **情绪价值** | 成就感、掌控感、"原来我的东西这么好"的惊喜 |

### 2.2 目标市场

- **MVP 阶段**: 国内中文用户
- **V2 阶段**: 考虑国际化

## 3. 功能需求（MVP）

### 3.1 功能总览

| # | 功能模块 | 优先级 | 说明 |
|---|---------|--------|------|
| F1 | 用户认证 | P0 | 邮箱注册/登录 |
| F2 | 添加妙物 | P0 | 手动输入产品信息 + 上传照片 |
| F3 | 妙物展架 | P0 | 按分类浏览所有产品（主界面） |
| F4 | 每日夸夸 | P0 | 每天随机展示一件产品，引导写赞美 |
| F5 | 热爱值 | P0 | 每次夸赞增加产品"热爱值" |
| F6 | 连续打卡 | P1 | 连续夸赞天数记录 |
| F7 | 成就勋章 | P1 | 里程碑解锁 |
| F8 | 产品详情 | P0 | 查看单个产品的夸赞历史和热爱值 |

### 3.2 F1: 用户认证

**描述**：用户通过邮箱注册和登录系统。

**用户故事**：
- 作为新用户，我希望通过邮箱快速注册，以便开始使用妙物记。
- 作为已注册用户，我希望通过邮箱和密码登录，以便继续使用。

**验收标准 (AC)**：
- [ ] 用户可以通过邮箱 + 密码注册新账号
- [ ] 注册后自动跳转到妙物展架（主页）
- [ ] 用户可以登录已有账号
- [ ] 未登录用户访问主页时自动跳转到登录页
- [ ] 有"退出登录"功能

**喵呜互动**：登录成功后，喵呜叼着钥匙出现，播放"开门"动画。

### 3.3 F2: 添加妙物

**描述**：用户手动添加产品到自己的收藏中。

**用户故事**：
- 作为用户，我希望快速添加一件产品，以便开始记录和夸赞它。

**输入字段**：

| 字段 | 类型 | 是否必填 | 说明 |
|------|------|---------|------|
| 产品名称 | text | ✅ | 例："iPhone 15 Pro" |
| 品牌 | text | ❌ | 例："Apple" |
| 分类 | select | ✅ | 四选一（见3.4） |
| 产品图片 | file | ❌ | 上传一张图片，存储到 Supabase Storage |
| 描述 | textarea | ❌ | 简短描述这件产品 |
| 购买日期 | date | ❌ | 什么时候买的 |

**验收标准 (AC)**：
- [ ] 用户可以填写表单添加新产品
- [ ] 产品名称和分类为必填
- [ ] 支持上传产品图片（单张，最大 5MB）
- [ ] 没有上传图片时显示默认的分类占位图
- [ ] 添加成功后跳转回妙物展架，新产品出现在对应分类下

**喵呜互动**：添加产品时，喵呜好奇地凑过来嗅一嗅新物品。

### 3.4 产品分类体系

| 分类 ID | 中文名 | 图标隐喻 | 典型产品 |
|---------|--------|---------|---------|
| `electronics` | 电子产品 | 铜质齿轮 ⚙️ | 手机、耳机、平板、相机、智能手表 |
| `software` | 软件服务 | 铜质指南针 🧭 | 订阅软件、App、在线工具、云服务 |
| `clothing` | 服饰 | 铜质衣架 👔 | 衣服、鞋子、包包、手表、配饰 |
| `books` | 书籍 | 铜质书签 📖 | 纸质书、电子书、有声书、课程 |

### 3.5 F3: 妙物展架（主页）

**描述**：用户的产品收藏主界面，按分类展示所有产品。

**用户故事**：
- 作为用户，我希望在一个地方看到我所有的产品，方便管理和选择夸赞对象。

**页面元素**：
1. 顶部：欢迎语 + 用户头像 + 喵呜
2. 分类标签栏（全部 / 电子产品 / 软件服务 / 服饰 / 书籍）
3. 产品卡片网格（每张卡片显示：图片、名称、分类、热爱值进度）
4. 右下角："添加妙物"悬浮按钮
5. 底部：导航栏（展架 / 每日夸夸 / 成就）

**验收标准 (AC)**：
- [ ] 默认显示"全部"分类
- [ ] 点击分类标签可切换分类筛选
- [ ] 产品卡片显示产品图片、名称、热爱值进度条
- [ ] 点击产品卡片进入产品详情页
- [ ] 点击"添加妙物"按钮进入添加产品页
- [ ] 支持下拉刷新（移动端）
- [ ] 无产品时显示空状态（喵呜探头探脑说"快来添加第一件宝贝吧~"）

**喵呜互动**：喵呜蹲在展架旁，偶尔伸懒腰或舔爪子。

### 3.6 F4: 每日夸夸（核心功能）

**描述**：每天随机选择一件用户的产品，引导用户写下赞美和发现。

**用户故事**：
- 作为用户，我希望每天有一个有趣的提示来重新关注我已有的产品。

**交互流程**：
1. 进入"每日夸夸"页面
2. 百宝箱打开动画 → 随机展示一件产品
3. 喵呜问一个引导性问题（随机从预设池中选取）
4. 用户在文本框中写下夸赞（最少 10 个字）
5. 选择心情标签（开心/惊喜/感恩/骄傲）
6. 点击"提交夸赞"
7. 产品热爱值 +1，喵呜开心跳起来
8. 显示当前连续打卡天数

**预设引导问题池**（随机选取一个）：
```
- "说说这件宝贝最近让你开心的一件事？"
- "这件产品有什么功能是你最常用但没想过的？"
- "如果要向朋友推荐它，你会怎么描述？"
- "它在哪个时刻让你觉得'还好买了它'？"
- "用了这么久，你发现了什么隐藏技巧吗？"
- "这件产品和你的其他东西搭配使用效果怎么样？"
- "回想一下，它在什么关键时刻帮到了你？"
- "如果给它打个分，你会给几分？为什么？"
```

**验收标准 (AC)**：
- [ ] 进入页面自动随机选择一件产品（当日未夸赞过的优先）
- [ ] 显示百宝箱打开的动画效果
- [ ] 显示喵呜和引导问题
- [ ] 文本框最少输入 10 个字才能提交
- [ ] 必须选择一个心情标签
- [ ] 提交后热爱值 +1
- [ ] 提交后更新连续打卡记录
- [ ] 如果今天已经夸赞过，显示"今日已完成"状态 + 喵呜满足表情
- [ ] 如果用户没有任何产品，引导去添加

**喵呜互动**：
- 展示产品时：喵呜翻出一件宝贝，好奇表情
- 用户输入时：喵呜歪头听，尾巴摇摆
- 提交成功：喵呜开心跳起来

### 3.7 F5: 热爱值

**描述**：每件产品有一个"热爱值"分数，每次被夸赞 +1。

**展示方式**：
- 产品卡片上显示玻璃温度计/铜质仪表盘样式的进度
- 热爱值等级映射：

| 热爱值 | 等级名称 | 温度计颜色 |
|--------|---------|-----------|
| 0 | 初见 | 空（透明） |
| 1-5 | 相识 | 浅蓝 |
| 6-15 | 熟悉 | 绿色 |
| 16-30 | 喜欢 | 暖黄 |
| 31-50 | 热爱 | 橙色 |
| 51+ | 挚爱 | 红色（满） |

### 3.8 F6: 连续打卡

**描述**：记录用户连续每日夸赞的天数。

**验收标准 (AC)**：
- [ ] 每天完成至少一次夸赞算打卡成功
- [ ] 连续天数在首页和夸赞完成页展示
- [ ] 中断一天则连续天数归零，但保留历史最长记录
- [ ] 喵呜根据打卡天数获得装饰（围巾、帽子等）

### 3.9 F7: 成就勋章

**描述**：用户达到特定里程碑时解锁成就勋章。

**成就列表**：

| 类型 ID | 标题 | 解锁条件 | 喵呜反应 |
|---------|------|---------|---------|
| `first_praise` | 初识之喜 | 完成第一次夸赞 | 喵呜竖起大拇指 |
| `streak_7` | 七日之约 | 连续打卡 7 天 | 喵呜戴上小围巾 |
| `streak_30` | 月度挚友 | 连续打卡 30 天 | 喵呜戴上小帽子 |
| `products_5` | 小小收藏家 | 添加 5 件产品 | 喵呜得意洋洋 |
| `products_20` | 宝物猎人 | 添加 20 件产品 | 喵呜被宝物围绕 |
| `praise_10` | 赞美达人 | 累计 10 条夸赞 | 喵呜鼓掌 |
| `praise_50` | 热爱大师 | 累计 50 条夸赞 | 喵呜戴上皇冠 |
| `all_categories` | 全品类探索者 | 4 个分类都有产品 | 喵呜展开地图 |

**验收标准 (AC)**：
- [ ] 成就在满足条件时自动解锁
- [ ] 解锁时弹出通知 + 喵呜送勋章动画
- [ ] 成就页面展示所有勋章（已解锁高亮，未解锁灰色剪影）
- [ ] 每个成就显示解锁时间

### 3.10 F8: 产品详情

**描述**：查看单个产品的详细信息和夸赞历史。

**页面元素**：
1. 产品大图
2. 产品信息（名称、品牌、分类、购买日期）
3. 热爱值进度（大号仪表盘）
4. 夸赞历史列表（时间线展示）
5. 编辑/删除产品按钮

**验收标准 (AC)**：
- [ ] 显示产品完整信息
- [ ] 大号热爱值仪表盘
- [ ] 夸赞历史按时间倒序展示
- [ ] 支持编辑产品信息
- [ ] 支持删除产品（二次确认）
- [ ] 删除后自动返回妙物展架

## 4. 非功能需求

### 4.1 性能

- 首屏加载时间 < 3s
- 页面切换动画 60fps
- 图片上传支持最大 5MB，压缩后存储

### 4.2 兼容性

- 移动端优先（断点：375px 起）
- 桌面端自适应
- 支持 Chrome、Safari、Firefox 最新版

### 4.3 安全性

- Supabase RLS 确保用户只能访问自己的数据
- 图片上传验证文件类型和大小
- XSS 防护
